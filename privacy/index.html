POPYANG Privacy Policy

Last Updated: April 5, 2026

Welcome to POPYANG (hereinafter referred to as "the Game"). We value your privacy and are committed to protecting the rights of creators. This document explains how we handle data and related licensing information.

1. Information Collection and Use

When you use the Game, we collect the following information through Google Firebase services:

  • Account Information: If you register using Google Login or email, the system records your email address and display name to create an account.
  • Game Data: We record the number of clicks (score) and store it in our database for the "Global Leaderboard" feature.
  • Device Information: Browser type or device model may be recorded solely to optimize the gaming experience.

2. Data Storage and Security

We utilize Google Firebase's cloud database (Firestore) and authentication system (Auth).

  • Your password is encrypted; the developer cannot view it directly.
  • All data is used exclusively for game rankings and will not be used for advertising or third-party data sales.

3. Public Information

Your Display Name and Click Score will be publicly visible on the "Top 10" leaderboard within the Game for all users to see.

4. Your Rights

You may stop using the Game at any time. If you wish to delete your account information or score records, please contact the developer via the GitHub project page.

5. Third-Party Services

The Game integrates Google Firebase to handle account authentication and leaderboard data storage. Data processing is governed by Google's Privacy Policy.

6. Policy Updates

We may update this policy occasionally. We recommend checking this page periodically for the latest information.

7. Contact Us

If you have any questions regarding this policy, please contact the developer via GitHub (091cc).
